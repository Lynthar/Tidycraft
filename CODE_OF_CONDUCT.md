# Code of Conduct

Tidycraft is a small open-source project. We expect contributors and participants to act in good faith — be respectful in discussions, focus on the technical merit of ideas, and assume positive intent.

## Scope

This applies to all project spaces: GitHub issues, pull requests, discussions, commit messages, and any direct communication about the project.

## Standards

In short:

- Be respectful and constructive.
- Disagree about ideas, not people.
- Welcome newcomers; experienced contributors were beginners once.
- No personal attacks, discrimination, or harassment of any kind.
- Off-topic political / ideological debates belong elsewhere.

## Enforcement

If something feels wrong, contact the maintainer (file a private GitHub Security Advisory or open a Discussion if appropriate). The maintainer reserves the right to remove comments, lock threads, or block contributors who repeatedly violate these standards.

This is intentionally lightweight while the project is small. As Tidycraft grows we may adopt the full [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) — feel free to suggest the upgrade via an issue.
