# Tidycraft 竞品分析报告

## 概述

本报告对比分析 Tidycraft 与三个主流数字资产管理工具的功能差异，并提出改进建议。

### 对比工具
1. **ShareCreators** - 智能数字资产管理系统（中国TOP20游戏公司使用）
2. **Mudstack** - 游戏工作室管线自动化平台
3. **Perforce Helix Core/DAM** - 行业标准版本控制（19/20 AAA工作室使用）

## 二、Tidycraft 当前优势

### 1. 轻量级桌面应用
- 基于Tauri 2.0，安装包小、启动快
- 无需服务器部署，开箱即用
- 完全离线工作，无隐私担忧

### 2. 高性能资产浏览
- 虚拟滚动支持10,000+文件流畅浏览
- 增量扫描，快速响应

### 3. 游戏项目类型识别
- 自动识别 Unity/Unreal/Godot 项目
- 读取Unity GUID信息

### 4. 资产分析引擎
- 规则化资产检测
- 问题分级（错误/警告/信息）
- 可导出分析报告

### 5. 批量操作能力
- 8种批量重命名模式
- 批量标签管理
- 多选操作支持

### 6. 良好的本地化
- 完整中英文支持
- 界面友好

## 三、需要补充和完善的功能

### 🔴 高优先级（核心竞争力差距）

#### 1. AI智能搜索
**现状**: 仅支持文件名文本搜索
**差距**: 三个竞品都支持AI搜索，ShareCreators还支持图像搜索和自然语言搜索

**建议实现**:
```
- AI自动标签：使用视觉模型识别资产内容
- 相似图像搜索：上传图片查找相似资产
- 自然语言查询："找所有红色的角色贴图"
- 智能推荐：根据使用习惯推荐相关资产
```

#### 2. 富媒体预览
**现状**: 仅显示图片缩略图和元数据
**差距**: 竞品支持3D模型交互预览、视频播放、音频播放

**建议实现**:
```
- 3D模型预览器：支持旋转、缩放、材质切换
- 视频播放器：支持时间轴、帧预览
- 音频播放器：波形显示、播放控制
- 大图预览：支持缩放、平移
- 动画预览：循环播放骨骼动画
```

**前端组件需求**:
- `ModelViewer.tsx` - Three.js/Babylon.js 3D渲染
- `VideoPlayer.tsx` - 视频播放器组件
- `AudioPlayer.tsx` - 音频播放器组件
- `ImageLightbox.tsx` - 大图查看器

#### 3. 版本历史功能
**现状**: 仅显示Git当前状态
**差距**: 竞品支持完整版本历史、对比、回滚

**建议实现**:
```
- 版本时间线：可视化文件变更历史
- 版本对比：并排对比不同版本
- 快速回滚：一键恢复到历史版本
- 变更统计：谁在什么时候做了什么修改
```

**前端组件需求**:
- `VersionHistory.tsx` - 版本历史面板
- `DiffViewer.tsx` - 文件差异对比视图
- `VersionTimeline.tsx` - 时间线组件

### 🟡 中优先级（体验提升）

#### 4. 协作与审核功能
**现状**: 完全单机使用
**差距**: 竞品都有完善的协作工具

**建议实现**:
```
- 资产评论：在资产上添加评论和回复
- 画面标注：直接在预览图上标注问题
- 审核状态：待审核/已批准/需修改
- 任务列表：与资产关联的待办事项
```

**前端组件需求**:
- `CommentPanel.tsx` - 评论面板
- `AnnotationTool.tsx` - 画面标注工具
- `ReviewStatus.tsx` - 审核状态组件
- `TaskBoard.tsx` - 任务看板

#### 5. 工具集成
**现状**: 基本的项目类型检测
**差距**: 竞品深度集成各种创作工具

**建议实现**:
```
- Unity编辑器插件：直接从Unity浏览和导入
- Unreal插件：资产浏览器集成
- Photoshop/Blender扩展：一键上传资产
- API接口：供第三方工具调用
- Webhook：资产变更时触发自定义操作
```

#### 6. 更多文件格式支持
**现状**: ~15种主要格式
**差距**: 竞品支持100+格式，包括专业3D软件原生格式

**建议实现**:
```
- 3D工程文件：.ma/.mb (Maya), .max, .blend, .c4d
- 设计文件：.psd, .ai, .sketch, .fig
- 文档：.pdf, .docx, .xlsx
- 游戏特定：.uasset, .umap (Unreal), .prefab (Unity)
```

### 🟢 低优先级（高级功能）

#### 7. 权限与用户管理
**现状**: 无用户系统
**差距**: 竞品有完善的权限控制

**建议实现**:
```
- 用户账户系统
- 角色权限（管理员/编辑/查看）
- 项目级权限隔离
- 外部访问链接（有限时效）
```

#### 8. 云同步功能
**现状**: 完全本地
**差距**: 竞品支持云存储和同步

**建议实现**:
```
- 可选云备份
- 多设备同步
- 团队共享空间
- 远程访问
```

#### 9. 高级安全功能
**现状**: 无
**差距**: ShareCreators有区块链追踪

**建议实现**:
```
- 操作日志审计
- 异常行为告警
- 数据加密存储
- 访问统计
```

## 七、总结

### Tidycraft 核心定位
作为轻量级本地数字资产管理工具，Tidycraft 应聚焦于：
- **快速高效**：秒级扫描，流畅浏览
- **离线优先**：无需服务器，保护隐私
- **专注游戏**：深度支持游戏引擎资产

### 与竞品的差异化策略
| 竞品 | 定位 | Tidycraft差异化方向 |
|-----|------|-------------------|
| ShareCreators | 企业级SaaS | 免费开源 + 本地部署 |
| Mudstack | 云优先协作 | 离线优先 + 快速启动 |
| Perforce | 企业版本控制 | 轻量级 + 零配置 |

### 最关键的改进项
1. **富媒体预览** - 这是体验差距最大的地方
2. **版本历史** - 资产管理的核心需求
3. **高级搜索** - 提升查找效率

通过以上改进，Tidycraft 可以在保持轻量级优势的同时，提供更专业的数字资产管理体验。
